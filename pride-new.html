A running list of the pride flags that my shop offers jewelry for. Use the sidebar to quickly navigate to specific flags.

The most recently added flags are genderflux, gaybian, oriented aroace, and Ace in Grace's alternate asexual.

If there's no photo listed for the style you're interested, message me and I'll add one.

Not every flag is available in all jewelry styles. Feel free to message me if you'd like to request any missing flags in any specific style and I'll add them to my listings if I have the bead colors.

Message me on Etsy!
